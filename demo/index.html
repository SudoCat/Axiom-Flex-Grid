Flexbox Responsive Grid System

A fully fluid mobile-first flexbox grid system utilising data attributes, keeping your classnames clean and semantic without sacraficing either readability or development speed. Rapidly develop highly versatile layouts. Selectors available with either traditional grid fractions (6/12) or as reduced fractions (1/2). Complete control over your responsive settings. Clean and readable mobile selectors using @size syntax (e.g. 1/2@mobile 1/4@desktop). Extend at your leisure!

Find the package on Github or npm!

Made by Robin Neal, part of the Axiom Digital Collective.
